Detailed Description

This class represents an Asian average price call option with European exercise type.

The payoff of this option at the time of expiration is given by the formula

payoff = max(0, bar(S)T - K)

where K is the strike price, and bar(S)T is the arithmetic average

bar(S)T = ∑i=1nS(ti)

of the option's underlying asset price at the observation times ti ( i = 1,&#8230;, n), with tn = T, the time of expiration of the option.

Note that the initial value S(t0) = of the price process is not included in the calculation of the average. However it will be included if the user sets the first observation time t1 to be the same as the initial time t0.
