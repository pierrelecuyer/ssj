Detailed Description

This class implements random variate generators for the fatigue life distribution [19]  with location parameter \(\mu\), scale parameter \(\beta\) and shape parameter \(\gamma\).

The density function of this distribution is

\[ f(x) = \left[\frac{\sqrt{(x - \mu)/{\beta}} + \sqrt{{\beta}/{(x - \mu)}}}{2\gamma(x - \mu)}\right] \phi\left(\frac{\sqrt{{(x - \mu)}/{\beta}} - \sqrt{{\beta}/{(x - \mu)}}}{\gamma}\right), \qquad x > \mu\tag{fFatigueLife} \]

where \(\phi\) is the probability density of the standard normal distribution.
