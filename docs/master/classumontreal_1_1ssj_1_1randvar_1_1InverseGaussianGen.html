Detailed Description

This class implements random variate generators for the inverse Gaussian distribution with location parameter \(\mu> 0\) and scale parameter.

\(\lambda> 0\). The density function of this distribution is

\[ f(x) = \sqrt{\frac{\lambda}{2\pi x^3}}\; e^{{-\lambda(x - \mu)^2}/{(2\mu^2x)}} \qquad\mbox{for } x > 0. \tag{fInverseGaussian} \]
