Detailed Description

This class implements random variate generators for the inverse gamma distribution with shape parameter \(\alpha> 0\) and scale parameter.

\(\beta> 0\). The density function of this distribution is

\[ f(x) = \left\{\begin{array}{ll} \displaystyle\frac{\beta^{\alpha}}{ \Gamma(\alpha)} \frac{ e^{-\beta/ x}}{ x^{\alpha+ 1}} & \quad\mbox{for } x > 0 \\ 0 & \quad\mbox{otherwise,} \end{array} \right. \tag{finvgam} \]

where \(\Gamma\) is the gamma function.
