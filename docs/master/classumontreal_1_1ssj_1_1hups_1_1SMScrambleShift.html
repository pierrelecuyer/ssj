Detailed Description

This class implements a umontreal.ssj.hups.PointSetRandomization that performs a striped matrix scrambling [189] and adds a random digital shift.

Point set must be a umontreal.ssj.hups.DigitalNet or an IllegalArgumentException is thrown.
