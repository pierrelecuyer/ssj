Detailed Description

This class implements random variate generators for the (continuous) uniform distribution over the interval \((a,b)\), where \(a\) and.

\(b\) are real numbers with \(a < b\). The density is

\[ f(x) = 1/(b - a) \qquad\mbox{ for }a\le x\le b. \tag{funiform} \]

The (non-static) nextDouble method simply calls inverseF on the distribution.
