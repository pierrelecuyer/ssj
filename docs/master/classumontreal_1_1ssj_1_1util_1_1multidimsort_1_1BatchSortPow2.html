Detailed Description

This is a subclass of BatchSort for which the batch numbers.

\(n_j\) are always powers of 2. One has \(n_j = 2^{e_j}\) for all \(j\) and \(p = 2^e\), where the \(e_j\) and \(e\) are integers. 

    The batch numbers can only be specified via the proportion exponents

\(\alpha_0,\alpha_1,…,\alpha_{d-1}\). The \(e_j\) are computed as a function of the number \(n\) of objects to be sorted. This number \(n\) is first rounded up the nearest power of 2, say \(p = 2^e \ge n\). (Often, it should already be a power of 2.) Then the \(e_j\) are defined as \(e_0 = \lceil e \alpha_0 \rceil\) and \(e_j = \lceil(e - e_0 - \cdots- e_{j-1}) (\alpha_0 + \cdots+ \alpha_j) \rceil\) for \(1\le j < d\). This gives \(e_0 + \cdots+ e_{d-1} = e\).
