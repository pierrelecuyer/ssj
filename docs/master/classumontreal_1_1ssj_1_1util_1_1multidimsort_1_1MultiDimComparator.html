Detailed Description

This class is useful if one wishes to perform an ordinary one-dimensional sort on MultiDimComparable<T> objects based on a single coordinate.

\(j\), which is specified in the constructor. It defines a bridge between the MultiDimComparable<T> interface and the classic Comparator in Java. It implements Comparator in a way that the method compare(o1, o2) compares two MultiDimComparable<T> objects in the dimension \(j\) given in the constructor, by calling o1.compareTo(o2, j).
